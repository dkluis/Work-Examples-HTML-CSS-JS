The Drinking Game

Use developer console to test the code or a play a game

to add a player: persons.push(new Person('A Name'))

to view all players all info: show_persons()

Game Console
Console Commands:
  1. persons.push(new Person('A Name')) -> Adds a new player to list

  2. show_persons() -> Lists all the players in the game
Notes
  1. persons is a list of player which the call Persons randomly assign a Race to a player
    You have to give it a Name
    You can optionally assign it an experiences as well
    persons.push(new Person('A Name', 'have to exist experience'))
    The push.persons() command put the new player into the persons list of players

  2. show_persons() -> Lists all the player that are the list and thus in the game
